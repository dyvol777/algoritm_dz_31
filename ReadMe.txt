Данное приложение реализует решение 31 варианта

31.Вы устроились на работу специалистом по ИБ. Руководство поставило
задачу обеспечения ИБ в предприятии. Известно, что при повышении
безопасности страдает удобство пользователей. У вас есть набор средств
и мер ИБ, каждая из которых защищает от одной или нескольких угроз.
При этом каждое средство снижает удобство пользователя. Постройте
алгоритм выбора средств, которые с одной стороны защищают от всех
известных угроз, а с другой — наименее неудобны для пользователя.

решение данной задачи будет сделано на преобразовании данной задачи к задаче булева программирования. 
задача булева программирования решается методом ветвей и границ с использование аддитивного алгоритма Балаша

вход: первое число - количество угроз, второе число - количество средств.
после этого вводятся вектора средств (1 если защищает, 0 если нет), в конце - стоимость использования данного метода. 

задача сводится к достижению покрытия всех угроз за минимальную стоимость.

на выход передается вектор из нулей и единиц(взято средство или нет)
